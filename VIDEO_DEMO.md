# Guion para video demo

Objetivo: mostrar la web como un producto vendible para centros de estetica, no solo como una maqueta.

Duracion sugerida: 35 a 45 segundos.

## Estructura

1. Apertura: 0-5 segundos
   - Mostrar el hero.
   - Mensaje hablado: "Creamos una web premium para centros de estetica que convierte visitas en reservas."

2. Servicios: 5-15 segundos
   - Scroll hacia servicios destacados.
   - Mensaje hablado: "Los tratamientos se presentan con categoria, duracion, precio y descripcion clara para que el cliente decida rapido."

3. Conversion: 15-25 segundos
   - Mostrar CTA, WhatsApp y seccion de confianza.
   - Mensaje hablado: "La experiencia esta pensada para usuarios que llegan desde Instagram o Google y quieren agendar sin friccion."

4. Sistema futuro: 25-35 segundos
   - Mostrar bloque de landing, administracion y bot IA.
   - Mensaje hablado: "La web queda preparada para crecer con panel de reservas, pagos y un bot IA que responda consultas frecuentes."

5. Cierre: 35-45 segundos
   - Mostrar contacto.
   - Mensaje hablado: "Ideal para centros de estetica que quieren verse profesionales y automatizar su atencion paso a paso."

## Tomas recomendadas

- Desktop: hero completo con CTA.
- Mobile: boton flotante de WhatsApp.
- Scroll lento por servicios.
- Close-up del bloque "Sistema futuro".
- Final en pagina de contacto.
